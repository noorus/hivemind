# hivemind
Starcraft II AI

Some snippets of code inspired by Dave Churchill's [CommandCenter](https://github.com/davechurchill/CommandCenter) and Blizzard's [s2client-api](https://github.com/Blizzard/s2client-api) examples.

Implementation of [linear-time labeling contour-trace](https://www.iis.sinica.edu.tw/papers/fchang/1362-F.pdf) (F.Chang, C-.J-. Cheng, C.-J. Lu 2003) from [BlockoS](https://github.com/BlockoS/blob). Image dumping via [stb_image](https://github.com/nothings/stb/).
